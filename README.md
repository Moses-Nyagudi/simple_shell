# Simple Shell

This is a collaboration project for ALX software developers for its Low Level language Module project. It aims on testing the basic understanding of C Programming and Group work as well as teaching on how to think like a software engineer.

The shell should be able to : -

- [ ] Read User input.
- [ ] Interpret given command.
- [ ] Execute command.
